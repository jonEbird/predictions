SMS Opt-In Policy

Application: Predictions App (Sports Betting Pool)

Last Updated: January 1, 2025

Purpose

This application is a private sports predictions platform used by a closed group of friends to manage friendly betting pools for college football games. SMS notifications are used to send game reminders, results, and administrative messages to group members.

How Users Opt-In to SMS Notifications

Users explicitly opt-in to receive SMS messages through the following process:

  1. Account Creation: Users create an account within the application
  2. Profile Settings: Users navigate to their profile/settings page
  3. Enable SMS: Users toggle the "SMS Notifications" setting to ON
  4. Phone Number: Users provide a valid mobile phone number
  5. Confirmation: Users save their preferences, confirming their opt-in
Important: SMS notifications are OFF by default. Users must explicitly enable this feature and provide their phone number to receive any messages.

Types of SMS Messages Sent

Users who opt-in will receive:

  • Prediction Reminders: Notifications before game time to submit predictions
  • Game Results: Final scores and winner announcements
  • Administrative Messages: Important group announcements from administrators

How to Opt-Out

Users can opt-out of SMS notifications at any time using any of these methods:

  1. Profile Settings: Log in and disable "SMS Notifications" in their profile
  2. Reply STOP: Reply "STOP" to any SMS message to unsubscribe immediately
  3. Contact Admin: Contact the group administrator to be removed from SMS notifications

Frequency

Message frequency varies based on the football season schedule. Users can expect:

  • Approximately 1-3 messages per week during active season
  • No messages during off-season
  • Occasional administrative messages as needed

Privacy

Phone numbers are stored securely and used only for the purposes described above. We do not share phone numbers with third parties. This is a private application for a closed group of participants.

Message and Data Rates

Standard message and data rates may apply based on the user's mobile carrier plan. The application does not charge fees for SMS notifications.

Support

For questions about SMS notifications or to request removal from the messaging list, users can contact the group administrator through the application or reply STOP to any message.

This SMS opt-in policy governs the use of SMS notifications within the Predictions App, a private sports betting pool application. By enabling SMS notifications in your profile settings, you consent to receive messages as described above.
